### Dependencies
NodeJS

### Configuration
Open and edit the file config.js
        {
            "discord_token" : "TOKEN HERE",
            "command_prefix": "-",
            "admin_ids": ["ADMIN ID HERE"]
        }

### How to set up
Simply download as a zip
and execute node bot.js
